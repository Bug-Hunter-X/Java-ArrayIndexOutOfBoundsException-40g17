# Java ArrayIndexOutOfBoundsException Bug

This repository demonstrates a common Java error: the `ArrayIndexOutOfBoundsException`. The `Bug.java` file contains code that attempts to access an array element outside its valid index range, causing the exception.  The solution, found in `BugSolution.java`, corrects the loop condition to prevent this error.  This is a simple yet crucial example highlighting the importance of careful array index management in Java.

## How to reproduce the bug:

1. Compile `Bug.java`.
2. Run the compiled code.
3. Observe the `ArrayIndexOutOfBoundsException`.

## How to fix the bug:

1. Replace `Bug.java` with `BugSolution.java`.
2. Compile and run the corrected code.
3. Observe the correct output.
